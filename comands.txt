Dependencias en entorno profesional
    sudo apt install -y build-essential libssl-dev libffi-dev python3-dev

Verificar donde esta python y pip
    which python3
    which pip3

Verificar las instalaciones
    pip3 freeze

Entorno virtual
    sudo apt install -y python3-venv # Solo para instalar el ambiente virtual. Si se usa linux o WSL
    python3 -m venv env #Entorno en cada proyecto
    alias avenv="source envs/bin/activate"


Creación de entorno virtual en Windows:
    py -m venv venv
    activar entorno virtual - .\venv\Scripts\activate
    deactivate - desactivar entorno virtual
    alias avenv=.\venv\Scripts\activate - crear atajo para activar env.
